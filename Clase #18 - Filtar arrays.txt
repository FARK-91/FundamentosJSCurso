Clase #18 - Maps y Filtar arrays.txt

Tenemos claro que un array es un conjuntos de valores posicionados.

para poder acceder a esos valores eventualmente necesitamos iterar sobre el array, en este caso tenemos un apoyo con la funcion filter(), donde ducha funcion recibe como parametro una condicion SIEMPRE, el resultado que nos devuelve son los valores dentro del array que coincidan con la condicion.

Sintaxis

var nombre_variable = nombre_array.filter(condicion)

Ejemplo:

var renier = {
	nombre : 'Renier',
	apellido : 'Ferrer',
	altura : '1.75',

}

personas = [renier]

var esMayor = ({ altura }) => altura > 1.8

var result = personas.filter(esMayor(renier))

// Funcion MAP, esta funcion lo que hace es modificar el array actual y devolver un nuevo array. Siempre recibe como parametro una funcion.

Sintaxis:

var nombre_variable = nombre_array.map(function)

Ejemplo:

var renier = {
	nombre : 'Renier',
	apellido : 'Ferrer',
	altura : '1.75',

}

personas = [renier]

// PODEMOS DEFINIR EL RETURN DE UN ARROW FUNCTION SIMPLEMENTE COLOCANDO LAS LLAVES QUE ENCIERRAN EL CUERPO DEL ARROW ENTRE PARENTESIS..... LOS PARETESIS DEFINEN EL RETURN

var alturaACms = persona => ({
	// Desglosamos el objeto para que no altere sus valores para siempre.
	...persona
	persona.altura *= 100

})

var result = personas.map(alturaACms)
